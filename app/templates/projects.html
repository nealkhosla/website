For this project, I worked in a team of 3 to develop an engine for an image-based shoe search. Using a combination of concepts from Computer Vision and Machine Learning, we were able to create a recommendation engine that would take in an image of a shoe and spit out 10 similar shoes. For our dataset, we scraped zappos.com database of approximately 13,000 men's shoes. Below I've included some images of example outputs from our algorithm. I hope to continue working on this project to make it more robust to orientation and image-diversity (zappos has a very nice dataset). If you're interested in discovering more about this project, feel free to contact me.


For this project, I used logistic regression to predict whether NFL teams would run or pass. I used a dataset that included play outcomes for the past 10 years in the NFL and extracted a feature set that included information on down, distance, time remaining in the game, score, and distance to score. Using a k-fold cross validation technique, I was able to achieve nearly 80% prediction accuracy. I hope to continue to build out this project to feature more team specific data and tendencies and achieve higher prediction accuracy (I may consider other models but the binary classification nature of this problem suggested to me that logistic regression was a good initial model to try and it worked fairly well). While I think I have a good initial start, with a little bit more leg work, I think such a project could be succesful.

This project wasn't necessarily a coding project but more of a game theoretic analysis I did for a class that was centered around determining optimal strategies for basketball teams in end-of-game situations. While the analysis certainly makes a number of simplifying assumptions, I think it demonstrates the right framework for thinking about this problem and definitely could be refined to properly provide insights into how to manage such situations. If you are interested in reading my very unpolished class paper that summarizes my methodology and results, you can find it here.

Recently, I built a web scraper using BeautifulSoup to scrape the website www.draftexpress.com and parse all of it's data on NBA drafts in the past 10 years. This involved pre-draft scouting data, including physical measurements from the NBA combine, college statistics, first-year NBA statistics, and career NBA statistics. For the statistics, I've included both basic and advanced stats as can be found on any player's stat profile on the website (e.g., Anthony Davis' page). Because the site had a number of inconsistencies in formatting and incomplete data, I had to throw out some of the data which you may find labeled with a "-1" value. However, after running the script, you should be able to do some post-processing and get it into a decent form for doing any sort of analysis. You can find the code for my scraper on my github.
